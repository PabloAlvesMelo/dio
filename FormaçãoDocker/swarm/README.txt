# Install docker swarm

Iniciar o docker swarm 
> docker swarm init

Adicionar uma VM a cloud do swarm, docker swarm join-token manager
> docker swarm join --token token ip:port

Listar nó ativos no cloud
> docker node ls 

Escluir nó do cloud
> 

comandos Vagrant

vagrant init 
vagrant global-status
